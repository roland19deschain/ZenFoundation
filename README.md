# ZenFoundation

A small, dependency-free library of concise, reusable extensions for the Swift standard library and Foundation.

## Highlights

- **Pure extensions** — no wrapper types, no executables, no resources, no dependencies.
- **UI-free** — usable on non-UI Apple platforms.
- **Foundation extensions:** `AttributedString`, `CharacterSet`, `DateFormatter`, `Decimal`, `FileManager`, `Locale`, `OperationQueue`, `NSPredicate`, `Scanner`, `URL`, `UserDefaults`.
- **Swift stdlib extensions:** `String`.

## Requirements

- Swift 5.10+ (Swift language version 5)
- iOS 14 · tvOS 14 · macOS 10.15 · watchOS 6

## Installation

**Swift Package Manager**

```swift
.package(url: "https://github.com/<owner>/ZenFoundation.git", from: "1.0.0")
```

Also distributed via **CocoaPods**.
